# Maquette Ohm Sweet Ohm

Projet de groupe réalisé lors d’une formation web.

Le site utilise **HTML** pour la structure et **Tailwind CSS** pour le style. On obtient ainsi une interface moderne, responsive et rapidement développée grâce à la collaboration des membres du groupe.  

## Figma 

https://www.figma.com/design/8uinWMBOoZmQT38p270jNH/Untitled?node-id=0-1&t=muEwQEbC41Vce7Ti-1

## Install

- npm init
- npm install tailwindcss @tailwindcss/cli
- Ajouter ce script à votre package.json : `"dev": "npx @tailwindcss/cli -i assets/lib/tailwind.css -o assets/css/style.css --watch"`
- npm run dev
